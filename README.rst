++++++++++++++++
Password Manager
++++++++++++++++

Presentation
============

Simple and secure password management.


Installation
============

.. code:: bash

    $ ghc pwdmanager.hs
    [1 of 1] Compiling Main             ( pwdmanager.hs, pwdmanager.o )
    Linking pwdmanager ...


Usage
=====

Add a new password:

.. code:: bash

    $ ./pwdmanager add passwd.txt serviceName userName password


List all entries:

.. code:: bash

    $ ./pwdmanager list passwd.txt
